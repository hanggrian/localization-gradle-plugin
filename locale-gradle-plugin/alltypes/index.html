All Types

com.hendraanggrian.locale.LocaleBuilder

Main interface used to configure localization content. All locales are based on https://github.com/umpirsky/locale-list/tree/master/data. Locales with code other than language and country code are excluded.
com.hendraanggrian.locale.LocaleExtension

Gradle extension to configure localization, any changes made here will take affect in WriteLocaleTask.
com.hendraanggrian.locale.LocalePlugin
com.hendraanggrian.locale.WriteAndroidResourcesTask

Task to write Android string resources files.
com.hendraanggrian.locale.WriteLocaleTask

Non-platform specific locale writer task.
com.hendraanggrian.locale.WriteResourceBundleTask

Task to write properties files which will then be used as java.util.ResourceBundle.
